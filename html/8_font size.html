font size

폰트 크기는 css의 font-size 속성을 사용하여 변경합니다.

폰트의 단위는 px, rem, em, vw, vh가 있습니다.

px(25px) : 가장 기본적인 단위

rem(2rem) : html 폰트 크기를 기준으로 배수에 해당하는 크기(기본 16px, 현 10px)

em(4em) : 부모 태그의 폰트 크기의 배수(body 태그의 4배)

span 태그의 텍스트

vw(5vw) : 화면 가로 크기의 1/100을 의미하는 단위입니다.

vw(5vw) : 화면 세로 크기의 1/100을 의미하는 단위입니다.
