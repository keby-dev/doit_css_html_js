font size

폰트 크기는 css의 font-size 속성을 사용하여 변경합니다.

폰트의 컬러는 color 속성을 사용하여 변경합니다.

폰트의 글꼴과 글꼴 유형은 font-family 속성을 사용합니다.

폰트의 굵기는 font-weight 속성을 사용합니다.

font-weight 속성은 해당 글꼴이 지원하는 글꼴의 굵기만 설정 가능합니다.

line-height 속성은 텍스트 높이를 변경할 수 있는 속성입니다.

보통 배수단위로 설정하나 px 단위로도 설정이 가능합니다.

다만 폰트의 크기보다 작을 시 글자가 겹쳐지는 오류가 생길 수 있습니다.
